核心功能

一个应用,三件事

我的 Skill

自动扫描所有工具的全部 skills 目录。按工具筛选、关键词搜索,网格 / 列表切换。一键卸载,或复制到其他工具。

Skill 市场

接入 skills.sh 全量列表(本地 24h 缓存)。搜索、分页,官方来源(anthropics / vercel-labs / microsoft)有标记,一键安装。

安装 Skill

三种方式都支持:分享短链、GitHub URL(owner/repo、完整链接、tree 链接均可)、上传本地 .zip(自动找到含 SKILL.md 的目录)。

常见问题

FAQ

只支持 macOS 吗?

目前 Skillkit 仅提供 macOS 版本。底层基于 Electron,未来可扩展到 Windows / Linux。

我的 skills 和数据存在哪?

skills 仍在各工具自己的目录(~/.claude/skills 等),应用数据存在 ~/Library/Application Support/skillkit/。Skillkit 不复制、不上传你的 skills——它只是一个本地管理界面。

从市场 / GitHub 安装安全吗?

安装的 skill 来自公开仓库(skills.sh 列表、GitHub)。Skillkit 只负责拉取 tarball / zip 并解压到对应目录,不执行任何 skill 内容。是否信任某个 skill,取决于它的来源仓库。

分享的 skill 会被谁看到?

分享链接是公开的——任何人凭链接即可访问并安装,7 天后自动过期。请勿分享包含敏感信息的 skill。

免费吗?

完全开源、免费。

skills.sh 是什么?

一个公开的 skills 市场。Skillkit 接入它的全量列表,你可以在应用内直接浏览、搜索、一键安装。
